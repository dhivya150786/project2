# project2
Project Overview

The Interactive Quiz Web Application provides an engaging platform for users to test their general knowledge through MCQs.
It features a clean UI, responsive design, and smart quiz flow including skipped question review and score evaluation.

Features

=> Start Screen

Welcome screen with a start button to begin the quiz.

=> Multiple Choice Questions

One question displayed at a time with four options.

=> Interactive UI

Hover effects and highlighted selected options.

Modern gradient background and smooth animations.

=> Skip Question Option

Users can skip questions and return to them later.

=> Review Skipped Questions

Prompt to answer skipped questions before final submission.

=> Score Calculation

Automatically calculates score based on correct answers.

=> Performance Feedback

Displays motivational messages based on the score achieved.

=> Responsive Design

Works on mobile, tablet, and desktop devices.

=> Touch-Friendly Controls

Optimized buttons and option selection for touch screens.

=> Pure Frontend Application

Built using only HTML, CSS, and JavaScript.

No external libraries or frameworks required.

** Technologies Used

HTML5 – Structure of the application

CSS3 – Styling, layout, and responsiveness

JavaScript (ES6) – Quiz logic, score calculation, and interactivity
